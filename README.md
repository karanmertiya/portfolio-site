# Modern Developer Portfolio

A high-performance, interactive portfolio website built with Next.js, TypeScript, and Tailwind CSS.

## 🌟 Features

- **Group-Based Organization**: Projects categorized by domain (Finance, Math, Game Theory, etc.)
- **Interactive Timeline**: Chronological journey of development milestones
- **Detailed Case Studies**: Deep dives with architecture diagrams, ML metrics, and code snippets
- **Glassmorphism Design**: Modern UI with smooth animations and gradients
- **Responsive**: Optimized for all devices

## 🛠️ Tech Stack

- **Framework**: Next.js 16 (App Router)
- **Styling**: Tailwind CSS v3
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **Type Safety**: TypeScript

## 🚀 Getting Started

1. **Install dependencies**:
   ```bash
   npm install
   ```

2. **Run development server**:
   ```bash
   npm run dev
   ```

3. **Build for production**:
   ```bash
   npm run build
   ```

## 📝 Customization

- **Projects**: Edit `data/projects.ts` to add your own projects.
- **Styles**: Modify `app/globals.css` to change the color theme.
- **Components**: Reusable components are in `components/`.
